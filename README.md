# 更新日志系统

一个简洁美观的更新日志管理系统，采用时间线设计风格。

## 功能特点

- ✨ 时间线风格的更新日志展示
- 📝 支持添加、编辑、删除更新日志
- 📎 支持上传附件（图片、视频、文档）
- 💾 智能存储：小文件存储在浏览器，大文件自动下载到本地
- 📊 支持 Excel 导入/导出
- 🖼️ 图片和视频预览功能
- 📱 响应式设计，支持移动端访问

## 文件说明

- `update-log.html` - 管理页面（用于添加、编辑、删除更新日志）
- `view.html` - 查看页面（只读模式，用于展示更新日志）

## 使用方法

### 管理员使用（添加日志）

1. 打开 `update-log.html` 文件
2. 点击右上角"添加更新日志"按钮
3. 填写更新日期、标题、重要更新和日常更新
4. 可选：上传附件（图片、视频、文档）
5. 点击"提交更新"

### 普通用户使用（查看日志）

1. 打开 `view.html` 文件
2. 浏览所有更新日志
3. 点击附件可以预览图片和视频

## 数据管理

### 导出数据
- 点击"导出 Excel"可以导出所有日志为 Excel 格式
- 导出的文件可以用 Excel 或 WPS 打开编辑

### 导入数据
- 点击"下载模板"获取 Excel 模板
- 在模板中填写更新日志
- 点击"导入 Excel"上传编辑好的文件

### 数据存储
- 小文件（<1MB）：自动存储在浏览器的 localStorage 中
- 大文件（≥1MB）：自动下载到本地，系统保存文件引用
- 建议定期导出数据备份

## 附件说明

### 支持的文件类型
- 图片：jpg, jpeg, png, gif
- 视频：mp4, avi, mov, wmv
- 文档：doc, docx

### 附件预览
- 图片：点击缩略图可以全屏预览
- 视频：页面内嵌播放器，可直接播放
- 本地文件：点击后选择本地保存的文件进行预览

## 技术栈

- 纯 HTML + CSS + JavaScript
- SheetJS (xlsx.js) - 用于 Excel 处理
- 无需服务器，可直接在浏览器中运行

## 浏览器要求

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## 注意事项

1. 数据存储在浏览器的 localStorage 中，清除浏览器缓存会导致数据丢失，请定期导出备份
2. localStorage 有大小限制（通常 5-10MB），建议定期清理旧数据
3. 大文件会自动下载到本地，请妥善保管这些文件
4. 本地文件预览时需要手动选择之前下载的文件
